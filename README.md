# 办公神器 (Bangong Shingqi) - 桌面娱乐与效率工具合集

一款专为办公环境设计的模块化桌面娱乐工具，支持一键隐藏、透明度调节及内置办公浏览器。

## 🚀 项目核心特性

- 🛠️ **模块化架构**：采用核心启动器（Core）与游戏模块（Games）分离的设计，支持无限扩展新游戏。
- 💼 **办公友好功能**：
  - **老板键 (Alt + Space)**：一键全透明并从任务栏消失。
  - **透明度调节 (Alt + Up/Down)**：支持多档透明度切换，从隐身到全显随心控制。
  - **内置浏览器 (Alt + B)**：在游戏界面内一键呼出 Bing 搜索，快速查阅资料或摸鱼。
  - **无缝切换 (Alt + Q)**：在游戏运行中随时返回主页，保留游戏状态。

## 🎮 当前合集游戏：星核防线 (Star Core Defense)

一款科幻风格的挂机塔防机器人战斗游戏。

### 游戏特色
- 🤖 **兵种系统**：包含主基地、采集工（Worker）、医疗兵（Healer）、游侠（Shooter）和守卫者（Guardian）。
- ☄️ **资源经营**：采集矿物（Minerals）赚取资金（Gold），升级兵种属性。
- 👾 **多样怪物**：包括史莱姆、蜘蛛、蝙蝠、蠕虫等多种形态，随着波次（Wave）增加难度。
- ⚡ **基地增援**：主基地拥有强大的 **6秒蓄力特效波**，能击退并伤害周围大范围敌人。
- 📊 **视觉反馈**：优化的基地血条显示、蓄力能量吸收粒子、击杀金币掉落及浮动伤害文字。

## 🕹️ 快捷键指南

### 通用（启动器与游戏内）
- **Alt + Space**: 切换老板键模式（显示/隐藏）
- **Alt + Up / Down**: 增加/降低窗口透明度
- **Alt + Q**: 从游戏返回主界面（不丢失进度）

### 游戏内专用
- **Alt + B**: 打开/关闭内置办公浏览器
- **Ctrl + R**: 重置当前游戏进度
- **鼠标左键**: 选择单位、点击控制面板进行升级/招募
- **鼠标右键**: 命令选中单位强制攻击目标点

## 🛠️ 技术实现

- **框架**: .NET 8.0 Windows Forms
- **渲染**: GDI+ 像素级双缓冲绘制，确保动画丝滑不闪烁。
- **Web 引擎**: 集成 Microsoft WebView2 核心，提供高性能内置浏览体验。
- **架构**: 采用基于组件的 Entity-Priority 系统。
