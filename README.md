# Pure Battle Game - 纯战斗游戏

从 CockroachPet 项目中提取的独立战斗游戏，移除了所有 AI、情绪、终端等无关功能，只保留核心战斗系统。

## 功能特性

- 🤖 **机器人战斗系统**
  - 自动移动与寻路
  - 多种攻击类型（激光、子弹、火箭等）
  - 近战格斗（陀螺模式）
  - 追逐与集火逻辑

- 👾 **怪物系统**
  - 高血量 Boss
  - 随机游走 AI
  - 被所有机器人集火攻击

- 💥 **投射物系统**
  - 7 种武器类型：BULLET, ROCKET, PLASMA, CANNON, LIGHTNING, SPIT, INK
  - 追踪机制
  - 特殊视觉效果

- 🎮 **游戏控制**
  - **Ctrl+N**: 添加新机器人
  - **Ctrl+M**: 开启/关闭怪物放置模式
  - **鼠标左键**: 选择机器人/怪物 或 放置怪物
  - **鼠标右键**: 让选中机器人发射子弹到鼠标位置
  - **Ctrl+R**: 重置游戏

## 快速开始

### 方法1: 使用 dotnet CLI

```bash
cd PureBattleGame
dotnet run
```

### 方法2: 使用 Visual Studio

1. 打开 `PureBattleGame.csproj`
2. 按 F5 运行

## 游戏玩法

1. **开始战斗**
   - 游戏启动后会自动生成 3 个机器人开始互斗
   - 机器人会自动互相攻击

2. **投放怪物**
   - 按 `Ctrl+M` 进入放置模式（鼠标变为十字）
   - 点击屏幕任意位置放置怪物
   - 所有机器人会立即转向攻击怪物

3. **添加更多机器人**
   - 按 `Ctrl+N` 添加新机器人
   - 机器人数量过多时会自动隐藏（超过5个）

4. **手动控制**
   - 鼠标左键点击机器人可以选中（黄色虚线框）
   - 选中机器人后，鼠标右键点击任意位置发射子弹

5. **游戏结束**
   - 只剩一个实体（机器人或怪物）存活时游戏结束
   - 5秒后自动重置

## 核心战斗逻辑

### 机器人行为优先级
1. **攻击怪物** (最高) - 如果有怪物目标，优先攻击怪物
2. **格斗模式** - 近距离与对手旋转碰撞
3. **追逐射击** - 中距离追逐并发射子弹
4. **随机移动** - 无目标时随机游走

### 攻击方式
- **普通机器人**: 慢射速，单发子弹
- **武器大师**: 快射速，多种弹药类型（火箭、等离子、加农炮等）

### 伤害与状态
- 基础伤害: 5-25 (根据武器类型)
- 状态效果: 击晕、减速
- 物理碰撞: 击退效果

## 技术实现

- **渲染**: GDI+ 双缓冲绘制
- **架构**: 简化版的 Entity-Component 模式
- **物理**: 简化的2D向量运动
- **AI**: 状态机 + 目标优先级系统
