# 纯战斗游戏 - 快速开始

## 游戏已就绪！🎮

### 运行方式

**方法1: 直接运行 (推荐)**
```
dotnet run
```

**方法2: 双击运行**
```
PureBattleGame\bin\Release\net8.0-windows\PureBattleGame.exe
```

### 操作控制

| 快捷键/操作 | 功能 |
|------------|------|
| **Ctrl + N** | 添加新机器人 |
| **Ctrl + M** | 开启/关闭怪物放置模式 |
| **鼠标左键** | 选择机器人/怪物 或 放置怪物 |
| **鼠标右键** | 选中机器人后发射子弹到鼠标位置 |
| **Ctrl + R** | 重置游戏 |
| **ESC** | 取消怪物放置模式 |

### 游戏玩法

1. **自动战斗**
   - 启动后自动生成 3 个机器人互斗
   - 机器人会自动追逐、射击、格斗

2. **投放怪物**
   - 按 `Ctrl+M` 进入放置模式（鼠标变十字）
   - 点击屏幕任意位置投放怪物（5000 HP）
   - 所有机器人会立即集火攻击怪物

3. **手动控制**
   - 鼠标左键点击机器人选中（黄色虚线框）
   - 选中后右键点击任意位置发射子弹

4. **胜负判定**
   - 只剩一个实体存活时游戏结束
   - 5秒后自动重置

### 战斗特性

- **机器人数量**: 无上限（超过5个自动隐藏）
- **怪物血量**: 5000 HP
- **机器人血量**: 1000 HP
- **武器类型**:
  - 普通机器人: 慢射速，单发子弹
  - 武器大师: 快射速，火箭/等离子/加农炮等多种弹药

### 技术信息

- **框架**: .NET 8.0 Windows Forms
- **渲染**: GDI+ 双缓冲
- **位置**: `PureBattleGame/`
- **所有代码**: 纯从 CockroachPet 提取，无AI/情绪/终端等无关功能
